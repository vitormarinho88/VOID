# 🖤 VOID – Marketplace Streetwear de Luxo

> Uma experiência de e-commerce premium desenvolvida em HTML, CSS e JavaScript puro.  
> Foco em design minimalista, dark mode, responsividade e simulação de carrinho de compras.



---

## 📋 Sobre o Projeto

**VOID** é uma aplicação front-end completa que simula um marketplace de roupas masculinas com estética **streetwear de luxo**. O projeto foi desenvolvido para compor portfólio de um desenvolvedor em transição de carreira, demonstrando domínio de:

- Layouts responsivos e modernos
- Microinterações e animações suaves
- Gerenciamento de estado (carrinho) com JavaScript puro
- Persistência de dados com `localStorage`
- Código limpo, sem dependências externas

A identidade visual segue os conceitos de marcas como **Fear of God**, **Supreme** e **Nike**, mas com personalidade própria: preto, branco, cinza e uma atmosfera exclusiva.

---

## ✨ Funcionalidades

- ✅ Catálogo de produtos com 10 itens da coleção **DROP 001** (t-shirts oversized, moletom, calça cargo, bomber jacket, acessórios)
- ✅ Adicionar/remover produtos do carrinho
- ✅ Ajustar quantidade de cada item dentro do carrinho
- ✅ Cálculo automático do total
- ✅ Sidebar do carrinho com animação suave
- ✅ Toast de notificação ao adicionar item
- ✅ Persistência do carrinho com `localStorage` (os itens permanecem após recarregar a página)
- ✅ Simulação de finalização de compra (alerta com resumo e limpeza do carrinho)
- ✅ Menu hambúrguer funcional no mobile
- ✅ Scroll suave para seções internas
- ✅ Design 100% responsivo (mobile, tablet, desktop)

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| **HTML5** | Estrutura semântica do documento |
| **CSS3** | Estilização moderna: Flexbox, Grid, variáveis CSS, backdrop-filter, transições |
| **JavaScript (ES6+)** | Manipulação do DOM, eventos, gerenciamento do estado do carrinho, localStorage |
| **Google Fonts** | Fonte *Inter* para tipografia limpa e legível |
| **Unsplash (imagens)** | Fotos de alta qualidade para representar os produtos (estilo streetwear) |

> ⚠️ Nenhum framework ou biblioteca externa foi utilizado. Tudo é JavaScript puro, garantindo leveza e performance.

## 🎨 Identidade Visual

- **Tipografia**: Inter (weights 300 a 700)

---

## 📱 Responsividade

- **Desktop**: grid de produtos com 4 colunas (ajustável até 1400px)
- **Tablet**: grid com 2-3 colunas, header adaptado
- **Mobile**: menu hambúrguer, grid com 1 coluna, sidebar do carrinho ocupando 100% da largura

---

## 🚀 Como Executar

1. **Clone ou baixe** este arquivo HTML.
2. **Abra diretamente** no seu navegador (não é necessário servidor).
3. Navegue pelos produtos, adicione ao carrinho, ajuste quantidades e finalize a compra simulada.

> Para editar, utilize qualquer editor de código (VS Code, Sublime, etc.). Todas as imagens são externas (Unsplash), portanto é necessário internet para carregá-las.
